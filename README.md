# 🏡 Property Price Prediction: Single-Variable Linear Regression

A foundational Machine Learning project that uses Simple Linear Regression to predict real estate prices based on property area. 

## 🚀 Overview
This repository demonstrates the core concepts of predictive modeling. By feeding a dataset of property sizes (in square feet) and their corresponding prices (in US$) into a Linear Regression algorithm, the model learns to calculate the "line of best fit." It then uses this mathematical relationship to automatically predict the market value of new, unlisted properties.

## 🛠️ Tech Stack
* **Language:** Python
* **Data Handling:** Pandas
* **Data Visualization:** Matplotlib (Scatter plots & Regression lines)
* **Machine Learning:** Scikit-Learn (`linear_model.LinearRegression`)

## 📊 Key Features
* **Exploratory Data Analysis:** Visualizes the raw data using scatter plots to confirm a linear relationship between area and price.
* **Model Training:** Calculates the optimal slope and intercept to fit a predictive regression line through the data points.
* **Batch Predictions:** Takes a new dataset of property areas, generates price predictions for all of them simultaneously, and successfully exports the results to a clean `prediction.csv` file.
